# Qubes OS 论坛爬虫 — 社区指南分区

**目标**：爬取 https://forum.qubes-os.org/c/guides/14 下所有帖子，合并输出为一个 markdown 文件。

---

## 1. 探索论坛结构

- [ ] 确认论坛是 Discourse，有标准 JSON API（`/c/guides/14.json`）
- [ ] 确认分页方式：`?page=N` 参数，翻到最后一页
- [ ] 确认单个帖子原始内容获取方式：`/raw/{topic_id}` 或 `/t/{slug}/{id}.json`

## 2. 写 Python 爬虫脚本（`crawl_guides.py`）

### 2.1 获取全部帖子列表
- 请求 `https://forum.qubes-os.org/c/guides/14.json?page=1`
- 解析 `topic_list.topics`，提取每个帖子的 `id`、`title`、`slug`
- 检查 `more_topics_url` 是否有下一页，循环翻页直到拿完

### 2.2 获取每个帖子正文
- 用 Discourse 原生 raw 接口：`https://forum.qubes-os.org/raw/{topic_id}`
- 只取第一个帖子（楼主帖），不要回帖
- 去掉 HTML 标签，保留纯文本/markdown 格式

### 2.3 去重 & 过滤
- 排除置顶帖（pinned）、分类说明帖（如 "About the Community Guides category"）
- 按帖子 ID 去重

## 3. 合并输出

- [ ] 所有帖子写入单个 `qubes-guides.md`
- 格式：
  ```
  # Qubes OS 社区指南汇总

  ## 帖子标题1
  正文内容...

  ---

  ## 帖子标题2
  正文内容...
  ```
- [ ] 不下载图片（图片链接保留原 URL 或直接去掉）

## 4. 验证

- [ ] 运行脚本，确认所有帖子都被爬到
- [ ] 检查输出 markdown 可读性
- [ ] 统计帖子数量与论坛显示一致
